通过 SSH 隧道链接远程 MySQL

Posted on 2018-10-01

场景：

  • SSH 跳板机：10.0.0.1
  • MySQL 数据库：10.0.0.2:3306

MySQL 部署的机器不允许 SSH 连接，并且数据库端口只允许 localhost 或 10.0.0.1 这台跳板机访问，此时本机可以通过 ssh 连上跳板机，通过 ssh 建立的隧道端口转发连接上 MySQL 数据库。
